# Udacity Self-Driving Car Nanodegree Projects
This repository contains a few selected completed projects in Udacity self-driving car nanodegree program ([link](https://www.udacity.com/course/self-driving-car-engineer-nanodegree--nd013)). The projects are related to perception, SLAM, path planning and control.

## Term1 - Perception

| *Project Description* | *Project Link* |
|:--|:--:|
| **Semantic Segmentation**: Deletecting road pixels in an image using Deep Learning based semantic segmentation | [Link](./CarND-Semantic-Segmentation) |
| **Object Detection**: Deletecting & localizing vehicles on a road using Histogram of Oriented Gradients (**HOG**) and a linear **SVM** | [Link](./CarND-Vehicle-Detection) |
| **Object Detection**: Deletecting driving lane using **color gradients** | [Link](./CarND-Advanced-Lane-Lines) |
| **Behavioral cloning**: Behavioral cloning using a **CNN** in a simulated environment | [Link](./CarND-Behavioral-Cloning-P3) |
